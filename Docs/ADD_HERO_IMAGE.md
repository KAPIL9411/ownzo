# How to Add Your Hero Person Image

## Quick Steps

1. **Locate your image file**
   - You have a PNG image of a person holding a package and tablet
   - The image should ideally have a transparent background

2. **Save it to the correct location**
   ```bash
   # Copy your image to:
   /Users/pradeepkumar/Ownzo/public/images/hero/person-with-package.png
   ```

3. **Using Finder (GUI method)**
   - Open Finder
   - Navigate to: `Ownzo/public/images/hero/`
   - Drag and drop your image file there
   - Rename it to: `person-with-package.png`

4. **Using Terminal (CLI method)**
   ```bash
   # From your Downloads folder (or wherever your image is):
   cp ~/Downloads/your-image.png /Users/pradeepkumar/Ownzo/public/images/hero/person-with-package.png
   ```

5. **Verify the image**
   - The file should be at: `/Users/pradeepkumar/Ownzo/public/images/hero/person-with-package.png`
   - Refresh your browser - the image will appear automatically!
